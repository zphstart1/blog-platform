## 注意事项

1. **Dockerfile 为单阶段构建**：由于 Docker Hub 不可用 (OPS-001)，Dockerfile 改为使用 `maven:3.9-eclipse-temurin-17` 单阶段构建（构建+运行在同一镜像），镜像体积较大。网络恢复后应改回多阶段构建。
2. **@MapperScan 配置**：`BlogApplication.java` 中显式扫描 DDD 三个 infrastructure 包，旧 `com.blog.mapper` 包下的 Mapper 通过 `@Mapper` 注解自动发现。
3. **ArticleTagPO 警告**：`ArticleTagPO` 缺少 `@TableId` 注解，MyBatis-Plus 的 `ById` 方法不可用。目前不影响功能。
4. **DataInitializer**：已改为使用 DDD 的 `UserMapper` 和 `UserPO`。
5. **前端 dist**：blog-admin 使用已有 dist，未重新构建。如前端代码有变更需重新 `npm run build`。
